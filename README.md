# 🏆 RaizeStudfy – Lộ Trình Học Java Từ Newbie → Pro

## Mục Tiêu

Nắm vững **toàn bộ Java** – từ syntax cơ bản → OOP → Collections → Concurrency → JVM Internals → Modern Java → Ecosystem. Học đi đôi với thực hành: mỗi bài có lý thuyết, ví dụ trực quan và bài tập thực chiến.

## 📌 Nguyên Tắc Học Hiệu Quả

1. **Đọc lý thuyết** → Hiểu khái niệm
2. **Chạy code mẫu** → Thấy kết quả thực tế
3. **Thay đổi, thử nghiệm** → Phá vỡ để hiểu sâu hơn
4. **Làm bài tập** → Củng cố kiến thức
5. **Build mini-project** → Apply vào thực tế

> 💡 **Tip:** Không cần hiểu 100% mọi thứ ngay. Đọc qua, code theo, làm bài tập, rồi về sau sẽ hiểu sâu hơn khi đọc lại.
